Second, set <%= browser %> as your default browser

  2. Click Microsoft Edge under the “Web browser” heading in the opened window
  3. Click <%= browser %> under “Choose an app”
  4. You can now Close the settings window

Edge Deflector is an open source application that forces Cortana to use your default browser (which now should be <%= browser %>).
